## iGEM 2025 CSMU-Taiwan 競賽網站

### 背景

iGEM（International Genetically Engineered Machine）為全球最大的國際合成生物學競賽，由麻省理工學院於 2004 年發起，每年吸引世界各地頂尖大學與高中組隊參賽。2025 年共有 421 支隊伍報名，參賽者需結合合成生物學、工程與計算機科學等跨領域能力，提出具實驗驗證的創新技術方案，以解決醫學、食品、環境等社會議題。

本屆中山醫學大學（CSMU-Taiwan）以抗 IL-23 抗體可變區分泌技術「Coliguard」為主題，研究藉由哺乳類細胞分泌抗體功能區域片段，用於治療潰瘍性結腸炎（UC），於 2025 年 10 月赴法國巴黎參加 Jamboree。
